# My Private Shopify

This is a minimal Next.js project scaffolded with TypeScript. It includes:
- A basic home page
- Public and styles folders
- Preserved SHOPIFY_UI_ROADMAP.md and config files

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run the development server:
   ```sh
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
